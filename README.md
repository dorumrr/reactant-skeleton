# ReactANT Skeleton

<em>I love React, and I wanted to learn a bit more about [Ant Design](https://ant.design/).</em>

### This skeleton's foundation:
- create-react-app
- react-router
- unstated
- Ant Design (customization ready)
- gh-pages

### INSTALLATION

`yarn install`

### DEVELOPMENT

`yarn start`

### DEPLOYMENT (Github Pages)

`yarn run deploy`
